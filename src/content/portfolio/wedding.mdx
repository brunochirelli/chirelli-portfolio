## Objetivos

O objetivo do projeto é fornecer aos convidados do nosso casamento, uma forma prática para confirmarem presença, obterem dados do evento e comprarem seu presente, tudo em um mesmo lugar.

O projeto está em desenvolvimento.

### Confirmação

Para a confirmação, o intuito é fornecer no convite um código QR individual que quando escaneado irá fazer login com os dados dos convidados para aquela família.

Lá a presença pode ser confirmada e em sequência poderá comprar um presente.

### Presente

Para facilitar a compra dos presentes e para que nós, noivos, recebamos não o produto, mas sim o dinheiro referente a ele, foi criado um e-commerce simples para que cada família compre o presente que desejar (apenas convidados podem comprar).

### Tecnologia

Foi utilizado React, Redux para o gerenciamento do state global. Para gerenciar o conteúdo, usamos um gerenciador de conteúdo chamado GraphCMS que fornece uma API robusta através de queries via GraphQL.
